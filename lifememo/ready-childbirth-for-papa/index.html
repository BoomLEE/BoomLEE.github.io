아빠의 출산준비

아빠의 출산준비 :땡콩과의 만남 3개월전

  • 아기옷
    • 배냇저고리
      • 순면으로 만들어져 피부를 보호
      • 상표는 옷 바깥쪽
      • 위 아래 4벌.
      • 계절 고려
    • 우주복
      • 보온효과 좋은 녀석으로
    • 내의
      • 상하의 나누어진 내의로 (기저귀 갈기가 편리함)
  • 젖병/젖꼭지
    • 우선 젖병과 젖꼭지는 소모품임.
    • 어느 정도 변색, 상태가 좋지 않을 때는 교체할 것
    • 대략 젖병은 6개월에 한번 교체
    • 젖꼭지는 2개월에 한번 교체
    • 젖병과 젖꼭지는 매일 삶아 소독 하고 말리기 ->> 요건 아이템 구입 : 시밀레 젖병 소독기?
  • 기저귀
    • 일자형
      • 신생아 때 무난하나, 활동이 활발해지면 옆으로 샙니다.
    • 팬티형
      • 생후 1-2개월 뒤부터 팬티형으로
      • 일자형 보다는 비쌈.
    • 기저귀는 아기마다 맞는 기저귀가 다르다고함. 처음에는 다양하게 써볼것.
    • 안맞는다는것 : 기저귀 갈아줄 때 엉덩이에 뭐가 날때.
  • 물티슈
    • 아기들 소변보고 난 후 주로 사용
    • 아기용품/아기주변 청소에도 사용
    • 응가는 물티슈보다는 샤워기.
    • 종류
      • 자극이 적은 손입전용 물티슈.
      • 소독기능이 강한 제균 물티슈
  • 목욕 용품
    • 매장 점원말만 듣고 사지 말 것.
    • 필요한 만큼 구매할 것
    • 용품
      • 비누
      • 샴푸
      • 로션/파우더/오일 : 땀띠, 발진, 짓무름을 방지.
    • 무엇을 사야할지 망설여지면 샘플을 이용해 볼 것
  • 아기띠
    • 태어난지 100일 뒤에 주로 사용
    • 나들이의 필수품
    • 아기의 무게를 잘 분산시켜주는 것이 중요함

강조

em

*또는_1개로 문자열을 감쌈.

This is *italic*.
This is _italic_.

This is italic. This is italic.

strong

*또는_2개로 문자열을 감쌈.

This is **Bold**.
This is __Bold__.

This is Bold. This is Bold.

em + strong

*또는_3개로 문자열을 감쌈.

This is ***italic&bBold***.
This is ___italic&bBold___.

This is italic&bBold. This is italic&bBold.
